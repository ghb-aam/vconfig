### `Vim Editor Configuration for CP`       

**Usages:**
1. Open Linux Terminal in `/home` directory,
2. Type `mkdir .vim`,
3. Press `Enter`,
4. Type `cd .vim`,
5. Press `Enter`,
6. Type `mkdir colors`,
7. Press `Enter`,
8. Type `cd colors`,
9. Press `Enter`,
10. Type `vim monokai.vim`,
11. Press `Enter`,
12. Press `i`,
13. Paste the [`monokai.vim`](https://github.com/mGH-aam/vConfig/blob/main/monokai.vim) file,
14. Press `Esc`,
15. Type `:wq`,
16. Press `Enter`,
17. Type `cd`,
18. Press `Enter`,
19. Type `vim .vimrc`,
20. Press `Enter`,
21. Press `i`,
22. Paste the [`.vimrc`](https://github.com/mGH-aam/vConfig/blob/main/.vimrc) file,
23. Press `Esc`,
24. Type `:wq`, and
25. Press `Enter`.

That's it!
